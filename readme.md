# Appmenu - a start menu like program menu for ubuntu

##Dependencies:
python-appindicator,python-gio,python-gmenu   
to install via apt,
```sudo apt-get install python-appindicator```    
```sudo apt-get install python-gio```
```sudo apt-get install python-gmenu```
